# Contributing

Thank you for your interest in Council of Reeds.

This project is shared as a practical workflow template. Suggestions, issues, and pull requests are welcome, but the maintainer may not actively monitor the repository or respond quickly.

## Before You Contribute

Please keep contributions:

- simple
- practical
- beginner-friendly
- compatible with the existing workflow
- free of private or sensitive data

Do not include:

- API keys
- private transcripts
- client materials
- proprietary project files
- personal information
- unpublished internal company documents

## Good Contributions

Helpful contributions include:

- clearer setup instructions
- bug fixes in install scripts
- improved command templates
- sanitized examples
- corrections to confusing wording
- platform-specific setup notes

## Pull Requests

If you open a pull request, please explain:

- what problem it solves
- which files changed
- whether it affects Claude Code, Codex, Gemini CLI, or only documentation
- how you tested it, if applicable

## Issues

Issues are useful for:

- unclear setup steps
- broken install instructions
- missing docs
- command problems
- suggested improvements

Response is not guaranteed.
