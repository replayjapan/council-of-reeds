# Round 1 Prompt

The PLAI should write the Round 1 instructions here before giving startup instructions to the Reeds.

Recommended Round 1 structure:

1. Restate the user's goal in your own words.
2. Review the source of truth and relevant context files.
3. Identify what matters most.
4. Identify risks, missing inputs, and implementation constraints.
5. Recommend a practical direction.
6. Do not create final artifacts yet unless asked.

Output files:

- Claude Code: `round-1/claude.md`
- Codex: `round-1/chatgpt.md`
- Gemini CLI: `round-1/gemini.md`
