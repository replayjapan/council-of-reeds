# Final Position Prompt

Use this when the PLAI decides the Council is ready for final positions.

Each Reed must:

1. Read `prompt.md`, `context/source-of-truth.md`, all round folders, and all PLAI notes.
2. Review the other Reeds by name.
3. Challenge any final issue that would affect the quality of the final output.
4. State the final recommendation.
5. State what should be included.
6. State what should be excluded.
7. List unresolved assumptions or user decisions.

Output files:

- Claude Code: `final-position/claude.md`
- Codex: `final-position/chatgpt.md`
- Gemini CLI: `final-position/gemini.md`
